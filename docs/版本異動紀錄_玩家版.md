# 版本異動紀錄(玩家版)

給玩家看的更新日誌,只講「玩起來有什麼不一樣」,不放技術細節。技術細節請看
`版本異動紀錄_技術版.md`。

**維護方式(往後每次改動都照此格式新增一筆,寫在最上面、新的在前):**

```
## <日期> <小版本 / 大版本>

- 條列,每項一句話講清楚玩家會感受到的變化
```

---

## 2026-07-07 小版本

- 道具/武器/防具/裝備清單每一列的底色調淡,原本太深看不清楚。
- 共用倉庫(背包／倉庫兩側)裡,角色職業穿不了的裝備、學不了的技能書,欄位會出現淡紅色外框提醒。

---

## 2026-07-07 大版本

- 同步原作者最新版本,遊戲本體邏輯與樣式跟上最新版,所有加掛版功能(舊版橫列式清單、共用倉庫可學標示等)都保留不受影響。

---

## 2026-07-07 小版本

- 道具/武器/防具/裝備清單恢復成舊版的橫列式外觀(圖示＋名稱＋狀態文字),不是新版的方格背包樣式,但保留新版所有功能(雙擊裝備/使用、排序選單、技能書視窗等都沒有退版)。
- 共用倉庫的「背包」那一側,現在也會顯示「(可學)」標示(原本只有「倉庫」那一側有標)。

---

## 2026-07-07 大版本(接手基準)

- 第一次幫這個專案建立版本控制,之後每次改動都能追蹤紀錄與回復,不再只靠資料夾備份。這項不影響遊戲畫面或玩法。
